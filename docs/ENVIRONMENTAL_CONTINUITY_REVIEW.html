Environmental Continuity + Unified Workspace Feel — Sprint Review

Shipped 2026-05-15 to enterprise.colaberry.ai (commit 842acde). Three surgical changes — persistent focus in the context bar, formalized surface-arrival timing with reduced-motion respect, and Home + System container rhythm aligned — turn the workspace from four stitched-together pages into one continuous operational environment. No new components. No motion libraries. No animation framework.

Spatial continuity didn't need a new shell architecture. It needed the operator's current operational context to ride along in the persistent bar at the top of every surface, and it needed the two main reading rooms to share visual cadence. Everything else — the route tree, the transition timing, the layout system — was already there.
What this sprint explicitly is NOT: cross-fade transitions, slide-up motion, parallax, cinematic UI, animation libraries (Framer Motion / GSAP / Lottie all out of scope), new persistent UI chrome, or any restructuring of the route tree or layout shell. The only motion that exists is the same 220ms fade that was already there — now it respects prefers-reduced-motion.

1The persistent operational context — focus travels across rooms

The persistent context bar at the top of every authenticated surface now carries one calm line: "shaping <Focus Domain>". The signal comes from useWorkspaceMemory().memory.lastBpDomainLabel — written when the operator engages a domain on System BPs, read everywhere else. The operator never loses orientation when moving rooms.

Context bar — project · surface · shaping Lead Intelligence · in-flight line · whisper · production · 2026-05-15
How the line behaves:
  • Hidden by default — operator hasn't engaged a domain yet, the bar reads exactly as before (project · surface · in-flight · whisper).
  • Appears after the first System BPs engagement — clicking a flow strip stop, expanding a row, or jumping via a relationship chip writes lastBpDomainLabel; the bar picks it up immediately.
  • Carried verbatim across Home / Critique / Blueprint / System — the bar lives in PortalLayout, so the signal travels without per-surface plumbing.
  • Width-capped at 220px with ellipsis — never wraps, never elbows the in-flight slot.
  • Hover tooltip discloses provenance — "Your operational focus, carried across surfaces from your last System engagement."
