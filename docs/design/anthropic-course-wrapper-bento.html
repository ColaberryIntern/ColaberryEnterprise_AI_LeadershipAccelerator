Handoff notes

  • Tokens only — colors via semantic aliases (--surface-card, --text-strong, --text-muted, --red-500, --blue-500), type via --fs-* / Roboto + Roboto Mono (numerals), shape via --radius-xl / --radius-pill, motion via --ease-out. No raw hex.
  • Components — Card (--hoverable, --accent), Badge (neutral meta, solid "Start here"), Button (primary featured CTA, ghost compact, white-on-cherry anchor CTA), .cb-i + RemixIcon.
  • Color logic — cherry is reserved for emphasis (top accent, "Start here", featured CTA, the path anchor); berry-blue carries the learning icon tiles; everything else neutral. Warm, not loud.
  • Hierarchy — featured = lowest course number ("Start here"). Switch to "next / in-progress" once progress data exists.
  • Responsive — 3-col bento → single column under 760px; anchor wraps its CTA. Touch targets ≥38–48px.
  • Fallbacks — 2 courses: featured + 1 compact + anchor. 1 course: featured only, no anchor.
  • A11y — CTA aria-label names the course + "opens in new tab"; icons aria-hidden; dark mode via data-theme="dark"; visible focus rings from the token layer.
