- **Target audience:** enterprise executives 35–60, paying for an architect-level certification.
- **No emojis, no jokes, no excessive enthusiasm in copy.**
- **No em-dashes** (system-wide preflight rule).
- Color palette + typography to be derived from existing `enterprise.colaberry.ai` design system (the `baseline-ui` skill defines the canonical palette + tokens).
