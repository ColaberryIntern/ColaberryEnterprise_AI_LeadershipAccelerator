Already live: the Curriculum list in Basecamp (project 47502609, list 9946468992) is rebuilt as 12 week-groups × a 5-item checklist (60 todos), each week mapped to its Anthropic course, deadlines staggered before launch, Ali co-signing Intensive 1. Scripts are on main and the VPS. This blueprint is for building the student-facing platform around it.

1. The system we are building

One synced platform where each student takes the week's Anthropic course, applies it to their own project through an AI advisor that turns an idea into requirements and a task list, builds it in Claude Code (pushing to GitHub where we verify progress), previews it live, assembles a portfolio, and does it all inside a Skool-style community.
